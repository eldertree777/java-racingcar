# 자동차 경주 게임
## [3단계 - 자동차 경주](https://edu.nextstep.camp/s/E42ziiPQ/ls/btefkF6S)

### 기능 목록
- UI
  - InputView
    - 차량의 대수를 입력한다.
    - 시도할 회수를 입력한다.
  - ResultView
    - 실행 결과 화면을 출력한다.
    - 차량들의 위치를 출력한다.
- Function
  - Car
    - [ ] 자동차의 초기 위치는 0이다.
    - [ ] 차량은 전진 기능
    - [ ] 입력값이 0~3은 정지, 4~9는 전진하는 기능
    - [ ] 차량의 위치를 반환 하는 기능
  - CarManager
    - [ ] 자동차들의 위치를 반환한다.
    - [ ] 자동차들의 전진을 시도한다.
